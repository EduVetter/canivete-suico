== Descrição ==
O Canivete Suíço é um plugin multifuncional para WordPress, projetado para facilitar a vida de administradores e desenvolvedores, reunindo diversas ferramentas essenciais em um só lugar. Com interface moderna, dark mode, integração com Elementor e Gutenberg, recursos de segurança, personalização e suporte multilíngue (português, inglês, espanhol, alemão e hebraico), ele é ideal para sites profissionais.

**Principais funcionalidades:**
- Contador de visitas (com edição via painel, shortcode, bloco Gutenberg e widget Elementor personalizável)
- Customização do logo da tela de login
- Alteração segura da URL do wp-admin
- Limite de caracteres em meta descrição e títulos
- Exibição do tempo de leitura
- Abrir links externos em nova guia
- Coluna "Último login" na tabela de usuários
- Reordenação de itens do menu admin
- Logotipo personalizado por página/post
- Layout responsivo, dark mode, cards, ícones e cabeçalho fixo
- Suporte a múltiplos módulos e fácil expansão
- Compatível com português, inglês, espanhol, alemão e hebraico (arquivos .po/.mo em /languages)

== Description (English) ==
Canivete Suíço is a multifunctional WordPress plugin designed to make life easier for administrators and developers by bringing together essential tools in one place. With a modern interface, dark mode, Elementor and Gutenberg integration, security and customization features, and full multilingual support (Portuguese, English, Spanish, German, Hebrew), it's ideal for professional websites.

**Main features:**
- Visit counter (with panel editing, shortcode, Gutenberg block, and customizable Elementor widget)
- Custom login screen logo
- Secure wp-admin URL change
- Character limit for meta description and titles
- Reading time display
- Open external links in a new tab
- "Last login" column in user table
- Admin menu item reordering
- Per-page/post custom logo
- Responsive layout, dark mode, cards, icons, fixed header
- Modular and easily expandable
- Fully translatable (Portuguese, English, Spanish, German, Hebrew) – .po/.mo files in /languages

== Instalação ==
1. Faça upload da pasta `canivete-suico` para o diretório `/wp-content/plugins/`.
2. Ative o plugin pelo menu 'Plugins' no WordPress.
3. Acesse o menu "Canivete Suíço" no painel para configurar e ativar/desativar módulos.

== Installation (English) ==
1. Upload the `canivete-suico` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the "Canivete Suíço" menu in the dashboard to configure and enable/disable modules.

== Instruções de Uso ==
- **Contador de Visitas:**
  - Use o widget Elementor "Contador de Visitas" para exibir e personalizar o contador.
  - Ou utilize o shortcode `[canivete_suico_visit_counter]` em posts, páginas ou widgets.
  - O valor pode ser editado manualmente no painel do plugin.
- **Customização de Login:**
  - Ative o módulo e envie seu logo personalizado nas configurações do plugin.
- **Alteração do wp-admin:**
  - Ative o módulo e defina a nova URL de acesso ao painel.
- **Outros módulos:**
  - Ative/desative conforme necessidade pelo painel do plugin.

== Usage Instructions (English) ==
- **Visit Counter:**
  - Use the Elementor widget "Visit Counter" to display and customize the counter.
  - Or use the shortcode `[canivete_suico_visit_counter]` in posts, pages, or widgets.
  - The value can be edited manually in the plugin panel.
- **Login Customization:**
  - Enable the module and upload your custom logo in the plugin settings.
- **Change wp-admin URL:**
  - Enable the module and set the new admin access URL.
- **Other modules:**
  - Enable/disable as needed via the plugin panel.

== FAQ ==
= O plugin é compatível com multisite? =
Sim, o Canivete Suíço foi testado em instalações multisite.

= Como restaurar a URL do wp-admin se esquecer? =
Desative o plugin via FTP ou renomeie a pasta do plugin para restaurar o acesso padrão.

= O contador de visitas é editável? =
Sim, pode ser editado manualmente pelo painel ou via Elementor.

= O plugin é compatível com outros construtores além do Elementor? =
Sim, também funciona com Gutenberg via shortcode/bloco.

= O plugin está disponível em outros idiomas? =
Sim, suporta português, inglês, espanhol, alemão e hebraico. Traduções podem ser gerenciadas na pasta /languages.

== FAQ (English) ==
= Is the plugin multisite compatible? =
Yes, Canivete Suíço has been tested on multisite installations.

= How to restore the wp-admin URL if I forget it? =
Deactivate the plugin via FTP or rename the plugin folder to restore default access.

= Is the visit counter editable? =
Yes, it can be edited manually in the panel or via Elementor.

= Is the plugin compatible with builders other than Elementor? =
Yes, it also works with Gutenberg via shortcode/block.

= Is the plugin available in other languages? =
Yes, it supports Portuguese, English, Spanish, German, and Hebrew. Translations are managed in the /languages folder.
